Data Sources

3 つのデータベースに対応 三大数据源 3 Integrated Databases

ギャルゲー・アニメ・アニソン専門 DB を一つのアプリで横断検索。 黄油、动漫、动画歌曲数据库，一个应用全覆盖。 Galgame, anime, and anisong databases — all in one app.

ErogameScape (批評空間)
ギャルゲー 黄油游戏 Galgame

ギャルゲー・エロゲーの音楽情報に特化した日本最大級のデータベース。SQL ベースの高精度検索に対応。 日本最大的美少女游戏音乐数据库，支持基于 SQL 的高精度查询。 Japan's largest galgame music database with SQL-powered precise search.

Bangumi (番組計画)
アニメ・ゲーム全般 动漫·游戏 Anime / Games

REST API 経由で日本語タイトルに完全対応。アニメ・ゲームの関連楽曲を双方向で検索可能。 通过 REST API 实现对日文标题的完整支持，可双向检索动漫和游戏关联曲目。 Full Japanese title support via REST API. Bidirectional search for anime and game soundtracks.

Anison.info
アニソン専門 动画歌曲专属 Anisong Specialist

OP/ED/IN/AR まで網羅したアニソン専門 DB。作品ページから全使用楽曲を一括取得できます。 涵盖 OP/ED/IN/AR 的动画歌曲专属数据库，可从作品页一次性获取所有使用曲目。 Comprehensive anisong DB covering OP/ED/IN/AR. Fetch all songs from a series in one lookup.

Features

シンプルで強力な機能 简单而强大 Simple Yet Powerful

🔄

双方向検索双向检索Bidirectional Search

「曲名 → 作品」「作品 → 全楽曲」の 6 種類のモードをドロップダウン一つで切り替え。6 种搜索模式，下拉菜单一键切换，支持「曲名→作品」和「作品→全曲」两个方向。6 search modes in one dropdown. Switch between Song→Work and Work→Songs instantly.

📋

ワンクリックコピー一键复制Click to Copy

テーブルの任意のセルをクリックするとクリップボードにコピー。視覚的フィードバック付き。点击表格中任意单元格即可复制内容，带有视觉反馈提示。Click any table cell to copy its content instantly, with visual feedback confirmation.

🌏

CN ミラー対応大陆镜像支持CN Mirror Support

チェックボックス一つで ErogameScape のリクエストを koko.kyara.top 経由に切替。中国本土ユーザーも快適に。勾选「CN镜像」复选框，EGS 请求自动转发至 koko.kyara.top，中国大陆用户也可畅用。One checkbox routes EGS requests via koko.kyara.top for mainland China users.

日本語ファジー検索日文模糊搜索Japanese Fuzzy Search
